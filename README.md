# Portfolio — Minh Le

Personal portfolio website. Product Manager — turning problems into shipped solutions.

**Live:** https://portfolio-kappa-ten-45wunrkzan.vercel.app

## Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.
